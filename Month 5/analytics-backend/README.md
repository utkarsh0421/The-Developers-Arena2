# Analytics Backend

Reactive Spring WebFlux application handling WebSocket streams and REST APIs.

## Features
- **Reactive Streams**: Uses Project Reactor (Flux/Mono) for non-blocking I/O.
- **WebSocket & SSE**: Real-time data streaming.
- **Elasticsearch Integration**: For advanced search capabilities.
- **MongoDB**: For persistent metric storage.

## Run
```bash
mvn spring-boot:run
```
