# ERP System (scaffold)

This repository is a scaffold for an enterprise ERP system.

Modules:
- erp-core
- erp-hr
- erp-inventory
- erp-accounting
- erp-api
- erp-web

Run (development):
- Build: mvn clean package
- Run: mvn -pl erp-api spring-boot:run

See docs/ for architecture and operational runbooks.
